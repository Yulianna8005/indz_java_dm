# Система управління рибальством

Об'єктно-орієнтований прототип системи управління рибальством, розроблений на мові Python.

## Опис проєкту

Система демонструє взаємодію основних компонентів:
- **Рибалка** (Fisherman) - планує та проводить риболовлю
- **Сенсори** (Sensor) - моніторять температуру та якість води
- **Еколог** (Ecologist) - аналізує умови водойми
- **Служба погоди** (WeatherService) - надає прогноз для планування
- **Журнал виловів** (CatchLog, CatchLogService) - зберігає записи про виловів у SQLite БД
- **Експедиція** (FishingTrip) - управляє рибальською експедицією

## Вимоги до системи

- Python 3.7+
- Вбудований модуль `sqlite3` (входить у Python)

## Встановлення та запуск

### Крок 1: Клонування репозиторію

```bash
git clone <repository-url>
cd indz_java_dm
```

### Крок 2: Запуск програми

Запустіть програму з корневої папки проєкту:

```bash
python application.py
```

Або з явним зазначенням шляху:

```bash
python3 application.py
```

### Крок 3: Вивід результатів

Програма виведе детальний журнал роботи всіх компонентів системи:
- Планування рибальської експедиції
- Отримання прогнозу погоди
- Перевірку карт глибин та точок кльову
- Моніторинг умов водойми через сенсори
- Реєстрацію виловів
- Екологічну оцінку

## Функціональність

### Основні операції

1. **Планування** - Рибалка перевіряє погоду, карти та точки кльову
2. **Моніторинг** - Сенсори вимірюють температуру та якість води
3. **Риболовля** - Реєстрація виловлених риб у журналі
4. **Збереження** - Дані автоматично зберігаються у SQLite базі
5. **Аналіз** - Еколог надає звіти про умови водойми

### База даних

Програма використовує **SQLite** для збереження записів виловів:
- Файл бази даних: `fishing.db` (створюється автоматично)
- Таблиця: `catches` з полями: id, fisherman_name, fish_species, weight, timestamp

## Архітектура

Система використовує:
- **Dependency Injection** - впровадження залежностей через конструктор та setter методи
- **Type Hints** - статична типізація для всіх функцій
- **Docstrings** - документація всіх класів та методів українською мовою
- **ООП принципи** - інкапсуляція, спадкування та поліморфізм
